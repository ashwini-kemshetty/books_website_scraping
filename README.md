# books_website_scraping

Website used to scrape the data is "http://books.toscrape.com/"

### Libraries used
- **requests** to get the content from the webpage
- **BeautifulSoup** to pull the data and parse the HTML and XML pages
- **pandas** to save the scrapped data into the CSV file

### Scrapped Data

Scrapped all the pages of website and saved the result in .CSV file (image_url, title of the book, price of the book)
